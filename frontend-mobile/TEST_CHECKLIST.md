```bash

npx react-native start

# Android测试

# iOS测试（需要macOS）
npx react-native run-ios
```


## 🔧 问题排查指南

### 常见问题解决
1. **依赖安装失败**
   - 使用Yarn替代npm
   - 清理npm缓存：`npm cache clean --force`
   - 删除node_modules重新安装

2. **构建失败**
   - 检查Android SDK版本
   - 验证iOS证书配置
   - 查看详细错误日志

3. **运行时错误**
   - 检查设备连接状态
   - 验证网络连接
   - 查看控制台错误信息
